# 腎臓くん日記

透析患者の食事管理と血圧管理を、ご家庭で毎日記録するためのアプリです。医療判断ではなく、日々の記録・傾向確認・意識づけを目的にしています。

## 公開方式

今後はHTMLファイル配布ではなく、GitHub Pagesで公開する前提です。

- `index.html`: GitHub Pagesのトップページ
- `manifest.webmanifest`: PWA設定
- `sw.js`: オフライン表示用サービスワーカー
- `icons/`: ホーム画面追加用アイコン

## 主な機能

- 妻が見る画面
- 食事管理
- 血圧管理
- 履歴・グラフ
- 設定
- 共有コード
- PWA対応
- iPhone Safari / Edge対応
- ホーム画面追加対応
- オフライン時に最後に読み込んだ画面を表示

## 保存方式

記録データはブラウザ内のローカル保存です。同じiPhone、同じブラウザで開くと記録が残ります。PWAとしてホーム画面に追加した場合も、同じアプリ領域に保存されます。

## GitHub Pages

GitHub Pagesではリポジトリのルートにある `index.html` が公開トップページになります。Pages設定は `main` ブランチのルート公開を想定しています。
