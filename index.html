クラウド同期案

第1候補: Googleスプレッドシート連携

家族で見やすく、あとから表として確認しやすい方法です。最初の同期先に向いています。

第2候補: Supabase / Firebase

スマホ2台で自動同期する本格運用向きです。ログインや権限設定が必要です。

第3候補: GitHub Pages + 外部DB

公開URLで使いやすい一方、個人データ保護の設計を慎重に行う必要があります。
